# ☕ Java 기반 PC방 관리 프로그램
> **창업프로젝트 2 — 개인 프로젝트**  
> 기존 C언어 버전의 기능을 Java 환경으로 개선 및 확장  
> 회원 관리, 결제, 사용 시간, 주문, 매출 통계를 처리하는 **Java Swing 기반 PC방 관리 프로그램**

---

## 📘 프로그램 개요
본 프로젝트는 **C언어로 개발했던 PC방 관리 시스템**을  
**Java 언어 및 GUI(Swing)** 기반으로 리뉴얼한 버전입니다.  

사용자와 관리자의 기능을 분리하여,  
회원 관리, 주문, 결제, 시간 추가, 매출 집계 등의 기능을 시각적으로 구현했습니다.  
또한 기존의 텍스트 파일 기반 데이터 구조를 유지하면서  
**객체지향 설계(OOP)** 와 **이벤트 기반 프로그래밍**을 적용했습니다.

---

## 🗂 주요 기능

### 👤 사용자 기능
| 기능 | 설명 |
|------|------|
| 회원가입 / 로그인 | 사용자 계정 생성 및 로그인 |
| 자리 선택 | 사용 가능한 좌석 선택 |
| 시간 추가 | 원하는 시간만큼 결제 후 사용 시간 연장 |
| 먹거리 추가 | 음식 메뉴 선택 및 주문 |
| 사용 종료 | 사용 종료 시 자동 로그아웃 및 시간 정산 |

---

### 🧑‍💼 관리자 기능
| 기능 | 설명 |
|------|------|
| 로그인 | 관리자 계정 인증 |
| 회원 등록 | 신규 회원 추가 |
| 회원 수정 / 찾기 | 회원 정보 검색 및 수정 |
| 회원 삭제 | 회원 정보 삭제 |
| 회원 시간 추가 | 특정 회원의 사용 시간 조정 |
| 주문 관리 | 주문 내역 확인 및 처리 |
| 매출 현황 | 기간별 매출 합계 및 통계 확인 |
